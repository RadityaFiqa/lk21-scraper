# lks21-scraper

## Installation
```
TERMUX
1. pkg install git && pkg install nodejs-lts
2. git clone https://github.com/areltiyan/lk21-scraper.git
3. cd lk21-scraper
4. npm install node-fetch && npm install chalk && npm install readline-sync && npm install cheerio
5. node .

WINDOWS
1. Install nodejs https://nodejs.org/en/
2. Download zip https://github.com/areltiyan/lk21-scraper
3. Extract dan Tahan SHIFT + KLIK KANAN 
4. Open command window here/Powershell
5. npm install node-fetch && npm install chalk && npm install readline-sync && npm install cheerio
6. node .
```
